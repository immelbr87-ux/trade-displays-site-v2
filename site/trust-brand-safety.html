Trust rules

Workshoped replacement for “Trust Boundaries.”

Include real photos, call out known defects, and disclose missing parts. Don’t imply warranty, manufacturer support, or “new” status unless it’s verifiably true.
Use Grade A / Grade B definitions consistently. If a key component is missing, disclose it explicitly in “What’s included” and “Known issues.”
Pickup windows are agreed in advance. Items should be staged in an accessible location when possible. Buyer is responsible for vehicle/handling unless otherwise stated.
Listings that misuse trademarks, suggest affiliation, or make unsupported performance/health claims should be revised or removed.

Brand compliance (now included here)

Use this section as the consolidated replacement for the deleted Brand Compliance page.

Allowed

  • Accurate product naming (model/finish) and honest condition descriptions
  • Original product photos of the actual floor model item you’re selling
  • Neutral phrases like “Showroom Floor Model” and “Sold as‑is”
  • Clear disclosure when packaging, manuals, or hardware is missing

Not allowed

  • Implying manufacturer warranty or “authorized” status unless contractually true
  • Using logos in a way that suggests endorsement, partnership, or sponsorship
  • Misrepresenting condition as “new” when it’s a floor model
  • Omitting known defects or missing core components

Brand‑safe listing language (recommended)

Use: “Showroom Floor Model — sold as‑is. May show minor wear from showroom traffic. See photos. Pickup in agreed window.”

Avoid: “Like new / guaranteed / includes warranty / endorsed / factory supported” unless verifiably true.
